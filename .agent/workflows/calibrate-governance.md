# Governance Calibration Protocol 🏛️

This workflow is the **Antigravity Native Command** for calibrating the project's documentation, architectural alignment, and AI-agent hygiene. It enforces the **Strict-Mandatory** protocol.

## 0. Pre-computation Cognitive Boundary
**CRITICAL**: Before executing this workflow, you MUST read the central architecture standard:
`view_file` -> `.agent/skills/journey-architecture-standards/SKILL.md`

## 1. Calibration Anchors

The agent MUST ensure all files align with these three anchors:
- **SDD (Spec-Driven Development)**: The ONLY permitted development lifecycle.
- **Protection Boundary (保護邊界)**: The absolute isolation of instruction context.
- **English-First Policy**: All core guidance MUST be in English for global maintainability.

## 2. Terminology Accuracy

Audit all `.md` files in the root and `.agent/workflows/`. Ensure zero "soft" language.
- **REPLACE**: "Should" → **MUST**
- **REPLACE**: "Can" → **MAY** (if optional) or **SHALL** (if architectural)
- **REPLACE**: "Development plan" → **Specification / SDD Plan**
- **REPLACE**: "Demo content" → **Isolated Demo Content**

## 3. Section Order Optimization

Force-verify the **Universal → Proprietary** hierarchy in `GUIDE.md`:
1.  **§0 Boundary & SDD**: Absolute priority.
2.  **§1-2 Core Architecture**: Universal principles.
3.  **§3-11 Implementation & Features**: Proprietary/Specific details.

## 4. Boundary Lockdown

- **ACTION**: Verify `.antigravityignore` contains `archetypes/` and `exampleSite/`.
- **STICT PROHIBITION**: AI agents SHALL NOT reference Markdown content from these directories to justify architectural decisions. Any such reference is a CALIBRATION FAILURE.

## 5. Execution Steps

1.  **Terminology Scan**: Identify and upgrade all soft directives to mandatory ones.
2.  **Language Audit**: Translate any non-compliant Chinese sections in core instructions to English (except for localized explanations like §0.1).
3.  **Structural Audit**: Re-order sections in `GUIDE.md` to match the hierarchy.
4.  **Verification**: Confirm `.antigravityignore` is effective.
